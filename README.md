# ⚛ Energía Nuclear

Plataforma web educativa e interactiva sobre energía nuclear. Explora la física, la historia y los datos globales de la fuente de energía más densa del universo.

> Proyecto desarrollado para **Talent Tech · 2026**  
> Autor: **Luis Vicente Albarracin**

---

## Vista previa

| Sección | Descripción |
|--------|-------------|
| 🔬 Física Nuclear | Fisión, fusión, radiactividad y E=mc² con fórmulas |
| ⚙️ Reactor PWR | Diagrama SVG interactivo con neutrones animados |
| 📊 Datos Globales | 4 gráficas con producción, mix energético y CO₂ |
| 📂 Análisis CSV | Importa tus propios datos y visualízalos al instante |
| 🌍 Mapa Mundial | Distribución de reactores en 33 países |
| 📅 Historia | Timeline desde 1938 hasta hoy |
| ☢️ Isótopos | Tabla de isótopos clave con propiedades |
| 🛡️ Seguridad | Comparativa de riesgos y proyecciones al 2050 |

---

## Tecnologías

- **HTML · CSS · JavaScript** — sin frameworks
- **[Chart.js 4.4.1](https://www.chartjs.org/)** — gráficas interactivas
- **[PapaParse 5.4.1](https://www.papaparse.com/)** — importación y parsing de CSV
- **Google Fonts** — Orbitron · Share Tech Mono · Exo 2

---

## Uso

No requiere instalación ni servidor. Clona el repositorio y abre `index.html` en tu navegador.

```bash
git clone https://github.com/tinecve/Talent-EnergiaNuclear.git
cd Talent-EnergiaNuclear
# Abre index.html en tu navegador
```

## Estructura

```
Talent-EnergiaNuclear/
├── index.html   # Estructura y 8 secciones de la SPA
├── script.js    # Lógica, datos y gráficas
└── styles.css   # Tema oscuro y animaciones
```
